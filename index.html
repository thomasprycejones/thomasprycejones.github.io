Hey Momo — AI Voice Assistant for Kids

Child-safe conversational AI assistant running on SoyMomo devices. Custom Geolocation Engine

Proprietary server-side positioning service inspired by Google's Geolocation API and Skyhook. Fuses GPS coordinates, Wi-Fi SSID fingerprints, and cell tower data with caching layers for performance. Serves as the unified location backend for all SoyMomo device types, with plans for Bluetooth-based indoor positioning.

Telecom Integration Layer

Direct API and SFTP integrations with Latin American telecom providers (Entel, Claro) and global eSIM platforms (Gigs) for programmatic SIM activation, data plan provisioning, and lifecycle management across multiple countries and carriers.

Internal Operations Platform

Full-stack CS and ops platform built with NestJS, Prisma, and React. Features AI-powered customer service chatbot, NPS feedback system, Zendesk ticket integration, e-commerce ops dashboards (Bsale + Jumpseller), and Stripe/Shopify webhook processing.

Device Fleet Telemetry

Real-time monitoring infrastructure collecting battery state, storage/memory usage, installed/running apps, screen time, Bluetooth status, network connectivity, and GNSS fix quality from tens of thousands of deployed devices. Includes a battery history dashboard and remote diagnostics tooling.
